📜 Ancient Scroll says:"A variable is a named container that stores a value. In Python, no type declaration needed — just assign!"
📚 Learn

In Python, variables are created when you assign a value using =


Python data types:

  • int — integers like 42
  • float — decimals like 3.14
  • str — text like "hello"
  • bool — True or False
pythonExample
# Variables in Python
name = "Warrior"
level = 5
health = 98.5
is_alive = True

print(f"Name: {name}, Level: {level}")
# Output: Name: Warrior, Level: 5
